# EstacionamientoCSharp-Terminal
Este repo incluye el código de las clases utilizadas vistas en el tutorial para el mismo ejercicio. Está pensado como ejemplo introductorio del paradigma POO o OOP.

Es un ejercicio básico, que incluye objetos del tipo Vehículo, Cliente. Utiliza buenas prácticas como tener parámetros privados y manipular los objetos mediante Getters y Setters, así como código comentado.

La versión actual (1.0) utiliza herencia, dónde se explica su uso con ejemplos como Persona, Empleado y Cliente. Es perfectamente escalable y está planeado para ser utilizado con ejercicios para gráficos.
